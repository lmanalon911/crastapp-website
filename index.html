Built to feel natural

Every part of Crast is built to feel like normal human activity, not some robotic loop.

Manual mode

Full mixer control

Open the control panel and set exact Mouse %, Keyboard %, and Overall % with sliders. There's also a Constant/Range toggle if you want some natural variation.

Browser sync

Chrome tab switching

Switches tabs in your real, multi-tab Chrome windows on whatever interval you want. Fully adjustable, or just turn it off in Manual mode.

Persistence

Remembers your setup

Manual mode's slider values save automatically and come right back the next time you launch, even after a full reboot.

One tap on/off

Global hotkeys

Switch levels, open the mixer, or shut everything off from anywhere. No need to go hunting for a window first. A tray icon always shows what's running.

Runs quietly

Hidden by default

No window pops up during normal use. Everything's controlled through hotkeys and a small tray icon.

Always current

Built-in updates

A licensed install can check for updates and grab the latest version in one click, right from the app.
